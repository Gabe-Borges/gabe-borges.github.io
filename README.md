# Portfólio Pessoal - Gabriel Borges

Repositório contendo o código-fonte do meu portfólio pessoal. Este projeto foi desenvolvido com o objetivo de centralizar e apresentar minhas habilidades como desenvolvedor com foco em Front-End, além de expor projetos recentes, informações acadêmicas e fornecer um canal de comunicação direto.

## Características do Projeto

* **Design Responsivo:** Layout adaptável para diferentes tamanhos de tela (desktops, tablets e smartphones) garantindo uma experiência de usuário consistente.
* **Navegação em Página Única (Single Page):** Utilização de âncoras (smooth scrolling) para navegação fluida entre as seções Sobre Mim, Currículo, Projetos e Contato.
* **Processamento Assíncrono de Formulário:** Implementação de formulário de contato utilizando JavaScript (Fetch API) e Formspree, permitindo o envio de mensagens sem recarregamento da página.
* **Arquitetura CSS Modular:** Estilização desenvolvida com foco em manutenibilidade, utilizando variáveis globais (`:root`) e princípios DRY (Don't Repeat Yourself).

## Tecnologias Utilizadas

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização avançada, animações de transição e flexbox/grid layouts.
* **JavaScript (ES6+):** Manipulação do DOM, interceptação de eventos de formulário e requisições HTTP.
* **Formspree:** Serviço utilizado para o roteamento e entrega dos emails do formulário de contato.
* **Devicon / SVG:** Utilização de ícones vetoriais otimizados para interface e representação de tecnologias.
